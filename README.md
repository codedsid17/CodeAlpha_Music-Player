# PixelGroove

PixelGroove is a retro-inspired music player built using HTML, CSS, and JavaScript. Designed with a pixel-art aesthetic and a vintage turntable interface, it delivers a nostalgic listening experience while maintaining a clean and modern user experience.

## ✨ Features

-  Play, pause, next, and previous controls
-  Animated vinyl record player
-  Search songs and artists
-  Browse music categories
-  Favorites section
-  Playlist support
-  Shuffle functionality
-  Responsive design
-  Pixel-art inspired interface
-  Clean monochrome retro aesthetic

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎨 Design Inspiration

PixelGroove draws inspiration from:

- Vintage record players
- Retro handheld devices
- Pixel-art aesthetics
- Minimalist music interfaces

The goal was to blend nostalgia with modern web design to create a unique and enjoyable music listening experience.
